## ğŸ’¡ Tips & Tricks

### Getting Best Results
- **Short words work best** (3-7 letters)
- **Adjust left margin** to center your text
- **Use fancy colors** for vibrant patterns
- **Slower animation** (100-300ms) looks smoother
- **Save your patterns** using the data box

### Troubleshooting
- **Can't see pattern**: Check left margin slider
- **Pattern too wide**: Use shorter text or reduce spacing
- **Animation too fast**: Increase speed slider value
- **Colors not showing**: Select different color scheme
- **Controls missing**: Refresh the GitHub page
