# Privacy Policy for LED Tabela

**Last Updated**: January 11, 2026

## Overview

LED Tabela ("the Extension") is committed to protecting your privacy. This privacy policy explains how we handle information when you use our Chrome extension.

## Information Collection

**We DO NOT collect, store, or transmit any personal information.**

The Extension:
- Does NOT collect any user data
- Does NOT track your browsing history
- Does NOT send any information to external servers
- Does NOT use analytics or tracking tools
- Does NOT share any information with third parties

## Local Storage

The Extension uses Chrome's local storage API only to:
- Save your preferred settings (color schemes, animation speed, etc.)
- Store custom patterns you create
- Remember your last used configuration

**All data is stored locally on your device and never leaves your computer.**

## Permissions

The Extension requires the following permissions:

### Storage Permission
- **Purpose**: Save your settings and custom patterns locally on your device
- **Scope**: Only local storage, no cloud or external storage

### Tabs Permission
- **Purpose**: Detect when you're viewing a GitHub profile page to enable the extension
- **Scope**: Only reads the URL to determine if you're on a GitHub profile

### Host Permission (*.github.com)
- **Purpose**: Modify the visual display of contribution charts on GitHub profile pages
- **Scope**: Only cosmetic changes to the contribution chart display
- **Note**: Does NOT modify your actual GitHub data or contributions

## Data Security

Since we don't collect any data, there is no data to secure or breach. All settings are stored locally using Chrome's built-in storage API, which is encrypted and secured by Chrome itself.

## Third-Party Services

The Extension does NOT:
- Use third-party analytics
- Include third-party advertising
- Connect to any external services
- Load resources from external domains

## Changes to GitHub

The Extension only makes cosmetic, client-side changes to the visual display of your GitHub contribution chart. It:
- Does NOT modify your actual GitHub contributions
- Does NOT access your GitHub account data
- Does NOT make any API calls to GitHub
- Does NOT affect how others see your profile

## Children's Privacy

The Extension does not collect any information from anyone, including children under 13.

## Changes to This Policy

We may update this privacy policy from time to time. Any changes will be posted in the extension listing and on our GitHub repository.
