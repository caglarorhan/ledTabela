üîß Part 3: Common Issues & Solutions

Issue 1: Extension Won't Load

Symptoms: Error message when loading unpacked
Solutions:
  • Check manifest.json syntax (use JSON validator)
  • Verify all icon files exist in correct location
  • Ensure file paths don't have ./ prefixes
  • Check for file permission issues

Issue 2: Icons Not Displaying

Symptoms: Blank or default icon in toolbar
Solutions:
  • Verify PNG files exist: led-tabela_16.png, led-tabela_24.png, etc.
  • Check file names match exactly (case-sensitive)
  • Reload extension after adding icons

Issue 3: Popup Not Opening

Symptoms: Clicking icon does nothing
Solutions:
  • Check popup.html exists and path is correct
  • Look for JavaScript errors in popup
  • Verify popup.js and itrs.js load correctly

Issue 4: Content Script Not Running

Symptoms: No console message on GitHub pages
Solutions:
  • Reload the GitHub page after loading extension
  • Check matches pattern in manifest
  • Verify host_permissions includes github.com
  • Check content script files exist and load

Issue 5: Message Passing Fails

Symptoms: Actions in popup don't affect content page
Solutions:
  • Ensure sendResponse is called in listeners
  • Check chrome.runtime.lastError in console
  • Verify active tab has content script loaded
  • Test on correct domain (github.com)
